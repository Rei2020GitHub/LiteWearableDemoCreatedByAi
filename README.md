# Huawei Lite Wearable 開発デモ：バイタル＆姿勢モニター

## 概要
本プロジェクトは、Huawei Lite Wearable (ArkUI.Lite) 開発のハードルを下げ、システムエンジニアやプログラマが直感的に開発プロセスを理解するためのデモアプリケーションです。

心拍数センサーと加速度センサーを活用し、リアルタイムのバイタル計測と姿勢アドバイザー機能を提供します。

## 背景・目的
Lite Wearableのアプリ開発は主に2B向けが主流であり、個人の開発者による情報が非常に少ないのが現状です。
「興味はあるが、何から手をつければいいかわからない」というエンジニアに向けて、JavaScriptの知識と**生成AIの力**を組み合わせることで、未経験からでも短期間で実機動作するアプリが開発可能であることを証明するために作成されました。

## 特徴
- **リアルタイム心拍数計測**: センサーから取得した数値を即座に反映。
- **姿勢判定ロジック**: 加速度センサーを用いた「腕の角度」による姿勢アドバイザー。
- **実機向け権限設計**: ウェアラブル特有の権限エラー（Code 602など）に対するユーザー誘導の実装。
- **マルチ画面遷移**: ウェアラブル向けルーティングの実装。

## 開発のポイント：生成AIの活用
本プロジェクトの最大の特徴は、**Lite Wearable開発が未経験であっても、生成AIを「伴走者」として活用することで、短期間でプロトタイプを完成させた**点にあります。

- 複雑なセンサーAPIの呼び出し
- ウェアラブル特有のライフサイクル管理
- 画面レイアウト（HML/CSS）の自動生成
- エラーコード（602等）のトラブルシューティング

これらはすべて、AIとの対話を通じて解決されました。エンジニアにとって、新しいプラットフォームへの挑戦がかつてないほど容易になっていることを示しています。

## 動作要件
- **開発環境**: DevEco Studio 3.0 以上
- **対応デバイス**: Huawei Watch GT シリーズ等の Lite Wearable デバイス
- **言語**: JavaScript (ArkUI.Lite パラダイム)

## 構成
- `pages/index`: メイン画面（心拍数・加速度表示）
- `pages/detail`: 詳細画面（姿勢アドバイザー）
- `common`: 画像リソース（heart.png, ok_mark.png, warning_mark.png）
